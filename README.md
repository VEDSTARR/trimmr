# 🔗 URL Shortener

A full-stack URL shortening web app built with **React.js, Supabase, and Tailwind CSS**. It allows users to create short links, track analytics, and share via QR codes with a clean and responsive UI.

---

## 🚀 Features

- 🔗 Shorten long URLs
- 📊 Click tracking with device & location insights
- 📱 QR code generation
- 🎨 Responsive UI (Tailwind CSS + Shadcn UI)
- 🔐 Supabase backend integration

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS, Shadcn UI  
- **Backend:** Supabase  
- **Tools:** QR Code Generator, Analytics  

---

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/VEDSTARR/trimmr.git

# Navigate to project
cd trimmr

# Install dependencies
npm install

# Run the development server
npm run dev
